### 1.4 HTML/CSS/JS的关系 - 浏览器执行JS简介

- HTML/CSS标记语言，这两个都是描述性语言
- JS脚本语言 - 编程类语言
- 浏览器分成两部分:
  - 渲染引擎: 用来解析HTML与CSS，俗称内核
  - JS引擎: 用来读取网页中的JS代码。浏览器本身并不会执行JS代码，而是由JS引擎来执行的。JS引擎执行代码时逐行解释每一句源码(转换为机器语言)，然后由计算机去执行，所以JS语言归为脚本语言，会逐行解释执行。
